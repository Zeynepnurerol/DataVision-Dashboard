ASELSAN Excel Dashboard

Bu projede, ASELSAN’a ait finansal veriler kullanılarak Excel üzerinde bir dashboard hazırlanmıştır. Dashboard, verilerin daha kolay analiz edilebilmesi ve görselleştirilmesi amacıyla oluşturulmuştur.

Dashboard içerisinde aşağıdaki bileşenler yer almaktadır:
	•	Pivot tablolar
	•	Grafikler ve veri görselleştirmeleri
	•	Filtreleme ve veri analizi araçları

Bu çalışma, Excel veri analizi ve dashboard oluşturma becerilerimi geliştirmek amacıyla hazırlanmıştır.

Kullanılan araç:
	•	Microsoft Excel
